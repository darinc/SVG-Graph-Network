Basic Graph Network Example

Simple Network Visualization

This example demonstrates the basic usage of the SVG Graph Network library. The graph shows a simple network with different types of nodes and relationships.

Interactions

  • Pan: Click and drag on empty space
  • Zoom: Use mouse wheel
  • Drag nodes: Click and drag any node
  • Filter: Double-click a node to focus on its neighborhood
  • Reset: Double-click empty space or use the reset button
